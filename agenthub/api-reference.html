    Modules

    AgentHub

    Provides functions to request changes in the state of the given agent, for voice and chat.

    AgentHub.Campaigns

    Provides functions to deal with the campaigns like fetching the campaigns associated with a given user.

    AgentHub.Campaigns.HTTP

    Implements the Campaigns behaviour by fetching data via http requests.

    AgentHub.Campaigns.Mock

    Implements the Campaigns behaviour by mocking the data. This module is intended to be used during development only.

    AgentHub.ConfigHelpers

    Provide functions to ease reading configuration from environment variables. This code has been copied from https://gitlab.com/code-stats/code-stats/-/blob/master/lib/code_stats/config_helpers.ex and adapted to satisfy credo's rules.
