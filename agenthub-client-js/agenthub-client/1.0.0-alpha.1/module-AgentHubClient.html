AgentHubClient

This is the online documentation of the JavaScript library that a frontend application can use to interact with the AgentHub service. The AgentHub service is part of the Olos Live solution and provides functionality to the agents like requesting pause and sending dispositions. It is very likely this library will be used by the general Olos Live JavaScript library, sometimes refered to as "SDK".

(static) AgentHubClient(wsURL)

Source:
  • index.js, line 30

Returns an AgentHub Client object that you can use to interact with the AgentHub service running in the given URL. The returned object exposes all the functions you see listed below in this documentation page.

Example
const agenthub = AgentHubClient("ws://localhost:4000/socket")
Parameters:
Name Type Description
wsURL string

The websocket URL where the AgentHub service is listening.
Throws:

An error if the URL is not informed.

Returns:

An object with all the functions listed below in this documentation page, like connect, disconnect, pause, etc...

Registers a callback function for the given event name.

Example

Parameters:
Name Type Description

The name of the event you want to subscribe. Available events:

  • disconnected
  • pauseResponse
  • idleResponse

The function that will be called when the event is fired.
