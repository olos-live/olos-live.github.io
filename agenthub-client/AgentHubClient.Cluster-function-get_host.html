    AgentHubClient.Cluster.get_host

    You're seeing just the function get_host, go back to AgentHubClient.Cluster module for more information.
    Link to this function

    get_host(node)

    View Source

    Returns the host part of the given node atom (like the one from Node.self/0) as a string.

    Examples 

    iex> AgentHubClient.Cluster.get_host :"agenthub1@127.0.0.1"
"127.0.0.1"

iex> AgentHubClient.Cluster.get_host :"nonode@nohost"
"nohost"
