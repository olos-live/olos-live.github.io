    AgentHubClient (AgentHub Client v1.0.0-alpha.2) View Source

    This is the main client module. It provides the top-level functions to interact with the AgentHub application, like querying the state of agents, fetching list of agents and sending messages to an agent.

    Link to this section Summary

    Functions

    count_by_voice_state(campaign_id)

    Returns the number of agents on each voice state that has at least one agent, i.e., it won't include a state that has no agents in that state.

    get_all()

    Returns a list of all agents that are logged in, no matter the campaign.

    get_all_in_campaign(campaign_id)

    Returns a list of all the agents that are logged in a given campaign.

    get_by_user_id(user_id)

    Returns the state of the agent given its user ID.
